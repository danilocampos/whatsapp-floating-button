📱 Botão WhatsApp Flutuante

Integre facilmente em seu site

1️⃣ Método Rápido (Recomendado)

Copie e cole este código antes do fechamento da tag </body>:

<!-- Font Awesome -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Estilos do Botão WhatsApp -->
<style>
  #wa_button { bottom: 25px; right: 0px; position: fixed; z-index: 999; }
  .img-circle { background-color: #25D366; box-sizing: content-box; }
  .circlephone { border: 2px solid #25D366; width: 150px; height: 150px; bottom: -25px; right: 10px; position: absolute; border-radius: 100%; opacity: .5; animation: circle-anim 2.4s infinite ease-in-out; }
  /* ... mais estilos ... */
</style>

<!-- Botão WhatsApp -->
<div>
  <a href="https://wa.me/SEU_NUMERO_AQUI" id="wa_button" target="_blank">
    <div class="circlephone"></div>
    <div class="circle-fill"></div>
    <div class="img-circle">
      <div class="img-circleblock"><i class="fab fa-whatsapp"></i></div>
    </div>
  </a>
</div>

⚠️ Importante!

Substitua SEU_NUMERO_AQUI pelo seu número de WhatsApp no formato internacional.

Exemplos:

  • 🇧🇷 Brasil: 55 + 11 + 98765-4321 = 5511987654321
  • 🇵🇹 Portugal: 351 + 912345678 = 351912345678
  • 🇺🇸 EUA: 1 + 202-555-1234 = 12025551234

2️⃣ Método com Script Externo

Se preferir usar um arquivo JavaScript separado:

Passo 1: Adicione no seu HTML

<script src="caminho/para/whatsapp-button.js"></script>

Passo 2: Com Configuração Personalizada

<script>
  window.whatsappConfig = {
    phoneNumber: '5511987654321',    // Seu número
    position: 'right',                            // 'left' ou 'right'
    message: 'Olá! Como posso ajudar?'  // Mensagem pré-preenchida
  };
</script>
<script src="caminho/para/whatsapp-button.js"></script>

🎨 Personalização

Mudar Posição

No CSS, altere right para left:

#wa_button {
  bottom: 25px;
  left: 0px;    /* Mude de 'right' para 'left' */
  position: fixed;
  z-index: 999;
}

Mudar Cor

Procure por #25D366 (verde WhatsApp) no CSS:

border: 2px solid #007AFF;  /* Azul */
background-color: #007AFF;
/* Repita para todas as classes */

Mudar Tamanho

Ajuste os valores de width e height:

.circle-fill {
  width: 80px;    /* Aumentado de 100px */
  height: 80px;  /* Aumentado de 100px */
  /* ... resto do CSS ... */
}
